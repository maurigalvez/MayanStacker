===============================================
LAYOUT SETTINGS
===============================================

Canvas:
- Render Mode: Screen Space - Overlay
- Canvas Scaler: Scale With Screen Size
- Reference Resolution: 1920 x 1080
- Match: 0.5 (Width/Height)

Vertical Layout Group (for button containers):
- Child Alignment: Middle Center
- Child Force Expand: Width ✓
- Spacing: 20
- Padding: 40-40-40-40

Horizontal Layout Group (for settings rows):
- Child Alignment: Middle Left
- Child Force Expand: Width ✓
- Spacing: 20
- Padding: 10-10-10-10

Grid Layout Group (for level buttons):
- Cell Size: 150 x 150
- Spacing: 20 x 20
- Start Corner: Upper Left
- Start Axis: Horizontal
- Child Alignment: Upper Left
- Constraint: Fixed Column Count (3 or 4)

===============================================
COMPONENT SETTINGS
===============================================

Button:
- Width: 300-400px
- Height: 80-100px
- Transition: Color Tint
- Normal Color: White
- Highlighted Color: Light Gray
- Pressed Color: Dark Gray
- Disabled Color: 50% opacity

Slider:
- Min Value: 0
- Max Value: 1 (for volume)
- Whole Numbers: Off
- Handle Size: 20
- Direction: Left to Right

Toggle:
- Is On: Off (default)
- Graphic: Checkmark image

Dropdown:
- Template: Default (auto-created)
- Item Text: TextMeshProUGUI
- Caption Text: "Select..."

===============================================
NOTES
===============================================

1. All panels except MainMenuPanel should start INACTIVE
2. Make sure to add Canvas and EventSystem
3. Use TextMeshPro for all text (not legacy Text)
4. All back buttons return to the main menu
5. Create level buttons based on how many levels you have
6. Assign ALL references in MainMenuManager and SettingsManager
7. Remember to add scenes to Build Settings!
8. Game mode buttons are now directly in the main menu (no separate selection panel)
